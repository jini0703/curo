# CURO — AI Mental Wellness & Interview Companion

CURO is an AI-powered conversational companion designed to support users through stress, anxiety, and interview preparation. It provides empathetic conversations, guided interaction, and personalized responses using modern AI models.

## Features

* AI companion for stress & anxiety support
* Interview practice mode
* Voice-enabled interaction
* Conversation history tracking
* User authentication with Google OAuth
* Secure cloud database integration
* Personalized AI personas (ARIA & LEO)

## Tech Stack

* Frontend: React + TypeScript + Vite
* Backend: Supabase
* Authentication: Google OAuth
* Database: PostgreSQL (Supabase)
* AI Integration: Groq API / LLM Models
* Styling: CSS / Modern UI

## Installation

Clone the repository:

```bash
git clone https://github.com/jini0703/curo.git
cd curo
npm install
npm run dev
```

## Environment Variables

Create a `.env` file and configure:

* Supabase credentials
* GROQ API key
* ElevenLabs API key (optional)
